# **App Name**: ProResume AI

## Core Features:

- AI-Powered Content Generation: Generate resume content based on user-provided career information and job role, including tailored sentences. Employs a tool for selecting relevant phrases and achievements.
- Keyword Optimization: Suggest industry-specific keywords to enhance resume visibility for applicant tracking systems (ATS).
- ATS Compatibility Check: Analyze the resume document for ATS compatibility, highlighting potential issues.
- Multiple Format Export: Enable exporting resumes in PDF, DOCX, and HTML formats.
- Visual Template Selection: Provide at least three distinct visual resume templates with customization options for fonts, colors, and layout.
- Job Description Matching: Analyze the similarity of the generated resume to job description that are uploaded or referenced by URL.
- User Edit Feedback Loop: Collect and incorporate user edits and preferences to refine future resume outputs, learning user stylistic preference from edited text.
- Generate Resume: Generate Resume according to the user input.

## Style Guidelines:

- Primary color: Dark cyan (#3498DB) for a professional and trustworthy feel.
- Background color: Very light cyan (#EBF5FB), nearly white.
- Accent color: Violet (#9B59B6) for CTAs and highlights.
- Body text: 'Inter', a grotesque sans-serif, with a modern and objective feel. Headline Text: 'Space Grotesk', a proportional sans-serif.
- Use professional, clean icons for sections and key skills.
- Maintain a clean and structured layout across all templates.
- Use subtle transitions and loading animations to enhance user experience.
